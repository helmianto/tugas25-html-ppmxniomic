Mengenal Semantic Elements Di HTML5

  • Articele

    • Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kita, tag article bisa kita gunakan dengan element header dan footer
    Karena tag article memiliki type block, artinya element ini memiliki lebar seusai dengan lebar browser jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog makak kita bisa menambahkan aside di bagian bawah postingan ini
    Karena aside memiliki sifat block maka lebarnya sesuai dengan page jika kita tidak menentuakn lebarnya.
